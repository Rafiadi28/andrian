## 💡 BEST PRACTICES

### DO ✅

- ✅ Use `form-modern-style.css` as primary stylesheet
- ✅ Follow class naming conventions (form-*)
- ✅ Wrap forms in proper containers
- ✅ Test on multiple devices before deploying
- ✅ Use CSS variables for consistency
- ✅ Reference example HTML for patterns
- ✅ Keep documentation updated
- ✅ Test accessibility (keyboard, screen readers)

### DON'T ❌

- ❌ Mix old class names with new ones
- ❌ Add inline styles (use classes instead)
- ❌ Forget form-group wrappers
- ❌ Skip responsive testing
- ❌ Remove CSS comments
- ❌ Override CSS variables incorrectly
- ❌ Deploy without testing
- ❌ Ignore accessibility requirements
